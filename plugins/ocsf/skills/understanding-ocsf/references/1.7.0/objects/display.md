# Display

The Display object contains information about the physical or virtual display connected to a computer system.

## Attributes

| Attribute | Type | Requirement |
|-----------|------|-------------|
| `color_depth` | integer_t | optional |
| `physical_height` | integer_t | optional |
| `physical_orientation` | integer_t | optional |
| `physical_width` | integer_t | optional |
| `scale_factor` | integer_t | optional |
