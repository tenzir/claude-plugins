# Encryption Details

Details about the encryption methodology utilized.

## Attributes

| Attribute | Type | Requirement |
|-----------|------|-------------|
| `algorithm` | string_t | optional |
| `algorithm_id` | integer_t | recommended |
| `key_length` | integer_t | optional |
| `key_uid` | string_t | optional |
| `type` | string_t | recommended |
