# CWE

The CWE object represents a weakness in a software system that can be exploited by a threat actor to perform an attack. The CWE object is based on the  [https://cwe.mitre.org/](Common Weakness Enumeration (CWE) catalog.

## Attributes

| Attribute | Type | Requirement |
|-----------|------|-------------|
| `caption` | string_t | optional |
| `src_url` | url_t | optional |
| `uid` | string_t | required |
