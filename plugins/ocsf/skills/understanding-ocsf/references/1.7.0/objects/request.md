# Request Elements

The Request Elements object describes characteristics of an API request.

## Attributes

| Attribute | Type | Requirement |
|-----------|------|-------------|
| `containers` | object (container) | optional |
| `data` | json_t | optional |
| `flags` | string_t | optional |
| `uid` | string_t | required |
