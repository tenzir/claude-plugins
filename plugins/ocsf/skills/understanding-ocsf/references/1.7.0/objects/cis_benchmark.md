# CIS Benchmark

The CIS Benchmark object describes best practices for securely configuring IT systems, software, networks, and cloud infrastructure as defined by the  [https://www.cisecurity.org/cis-benchmarks/](Center for Internet Security. See also  [https://www.cisecurity.org/insights/blog/getting-to-know-the-cis-benchmarks](Getting to Know the CIS Benchmarks.

## Attributes

| Attribute | Type | Requirement |
|-----------|------|-------------|
| `cis_controls` | object (cis_control) | recommended |
| `desc` | string_t | optional |
| `name` | string_t | required |
