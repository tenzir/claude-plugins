# Discovery Details

The Discovery Details object describes results of a discovery task/job.

## Attributes

| Attribute | Type | Requirement |
|-----------|------|-------------|
| `count` | integer_t | recommended |
| `occurrence_details` | object (occurrence_details) | optional |
| `occurrences` | object (occurrence_details) | optional |
| `type` | string_t | recommended |
| `value` | string_t | optional |
