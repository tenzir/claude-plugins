# CIS CSC

The CIS Critical Security Control (CSC) contains information as defined by the Center for Internet Security Critical Security Control  [https://www.cisecurity.org/controls]((CIS CSC). Prioritized set of actions to protect your organization and data from cyber-attack vectors.

## Attributes

| Attribute | Type | Requirement |
|-----------|------|-------------|
| `control` | string_t | required |
| `version` | string_t | recommended |
