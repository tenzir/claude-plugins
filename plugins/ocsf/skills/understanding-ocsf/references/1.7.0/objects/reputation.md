# Reputation

The Reputation object describes the reputation/risk score of an entity (e.g. device, user, domain).

## Attributes

| Attribute | Type | Requirement |
|-----------|------|-------------|
| `base_score` | float_t | required |
| `provider` | string_t | recommended |
| `score` | string_t | optional |
| `score_id` | integer_t | required |
