# Operating System (OS)

The Operating System (OS) object describes characteristics of an OS, such as Linux or Windows.

## Attributes

| Attribute | Type | Requirement |
|-----------|------|-------------|
| `build` | string_t | optional |
| `country` | string_t | optional |
| `cpe_name` | string_t | optional |
| `cpu_bits` | integer_t | optional |
| `edition` | string_t | optional |
| `kernel_release` | string_t | optional |
| `lang` | string_t | optional |
| `name` | string_t | required |
| `sp_name` | string_t | optional |
| `sp_ver` | integer_t | optional |
| `type` | string_t | optional |
| `type_id` | integer_t | required |
| `version` | string_t | optional |
