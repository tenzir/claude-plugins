# Git

Git workflows for Tenzir repositories.

## Features

- **Commit Message Writing Skill**: Auto-triggered guidance for writing clear, consistent git commit messages

## Installation

```
/plugin install git@tenzir
```

## How it works

### Commit Message Writing Skill

The `commit-message-writing` skill activates when committing changes. It provides guidance on:

- Commit message format (subject under 50 chars, body wrapped at 72)
- Writing style focused on user capability, not implementation
- Best practices for atomic, self-contained commits
